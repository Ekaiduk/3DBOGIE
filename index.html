การประยุกต์ใช้

นำความรู้ไปประยุกต์ใช้ในงานซ่อมบำรุงและวิศวกรรมจริง

1. ความเป็นมาและความสำคัญของโครงการ

 การเรียนรู้ระบบขนส่งทางรางมีบทบาทสำคัญต่อการพัฒนาอุตสาหกรรมคมนาคมในประเทศไทยโดยเฉพาะในด้านโครงสร้างพื้นฐาน และ การซ่อมบำรุง หนึ่งในส่วนประกอบหลักของรถไฟ คือ โบกี้ (Bogie) ซึ่งทำหน้าที่รับน้ำหนักและช่วยให้รถไฟเคลื่อนที่ได้อย่างมั่นคง อย่างไรก็ตามการเรียนรู้โครงสร้างของ Bogie จากภาพนิ่งยังไม่สามารถแสดงรายละเอียดได้ครบถ้วน

 คณะผู้จัดทำจึงมีแนวคิดพัฒนาโครงงาน 3D Bogie เพื่อสร้างแบบจำลองชิ้นส่วน Bogie 17 ส่วนในรูปแบบ สามมิติ และ นำเสนอผ่านระบบ QR CODE เพื่อให้ผู้เรียนสามารถแสกน และ ดู Model ได้ทุกมุมมองเป็นการยกระดับสื่อการเรียนรู้ให้มีความทันสมัย และ เข้าได้ง่ายมากขึ้น

2. วัตถุประสงค์ของโครงงาน

  • 2.1 เพื่อสร้างแบบจำลองสามมิติของชิ้นส่วนโบกี้ จำนวน 17 ชิ้น
  • 2.2 เพื่อพัฒนาระบบคิวอาร์โค้ดเชื่อมโยงกับแบบจำลองสามมิติ
  • 2.3 เพื่อให้ผู้เรียนสามารถเข้าถึงข้อมูลและสื่อการเรียนรู้เชิงเทคโนโลยีได้สะดวก

3. ขอบเขตของโครงงาน

  • 3.1 โครงงานนี้จำกัดเฉพาะการสร้างแบบจำลอง สามมิติ ของชิ้นส่วน Bogie จำนวน 17 ส่วน โดยใช้ Software Blender และนำเสนอด้วยระบบ Model–Viewer ผ่านเว็บไซต์ต้นแบบ
  • 3.2 ระบบ QR CODE สามารถแสกนได้ผ่าน Smart Phone และ เชื่อมโยงไปยังโมเดลแต่ละชิ้นงาน
  • 3.3 ไม่รวมถึงการจำลองระบบกลไกการเคลื่อนไหวของรถไฟจริง

4. ประโยชน์ที่คาดว่าจะได้รับ

  • 4.1 ผู้เรียนเข้าใจโครงสร้างและหน้าที่ของชิ้นส่วนโบกี้ได้อย่างถูกต้อง
  • 4.2 ได้รับสื่อการเรียนรู้ที่ทันสมัยและเข้าถึงได้สะดวก
  • 4.3 เสริมทักษะการใช้ซอฟต์แวร์สามมิติและเทคโนโลยีดิจิทัลให้กับผู้จัดทำ
  • 4.4 เป็นแนวทางในการพัฒนาสื่อการเรียนรู้สำหรับสาขาระบบขนส่งทางราง
